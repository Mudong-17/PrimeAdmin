# PrimeAdmin

## 简介

PrimeAdmin 是一个基于 Vue 3 和 PrimeVue 的现代管理模板。它旨在提供一个灵活、可定制的解决方案，适用于各种企业级应用。

## 技术栈

- Vue 3
- PrimeVue
- TypeScript
- Tailwind CSS
- Rspack
- Turbo
